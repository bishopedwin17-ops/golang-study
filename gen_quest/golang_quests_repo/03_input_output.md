# Quest 03 — Input & Output

## Objectives
- Read user input
- Format output

## Tasks
1. Ask the user for:
- name
- age

2. Print:
```text
Hello NAME, you are AGE years old.
```

3. Build a simple calculator:
- addition
- subtraction
- multiplication
- division

4. Build a BMI calculator.

## Bonus Challenges
- Handle decimal inputs
- Format outputs to 2 decimal places

## Expected Skills
- fmt.Scan
- User interaction
- Formatting
