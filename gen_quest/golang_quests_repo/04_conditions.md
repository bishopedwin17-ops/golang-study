# Quest 04 — Conditionals

## Objectives
- Learn if/else
- Learn switch statements

## Tasks
1. Check if a number is:
- positive
- negative
- zero

2. Create a grading system:
- A
- B
- C
- D
- F

3. Build an age restriction system.

4. Determine if a year is a leap year.

- Multiple switch cases

## Expected Skills
- Control flow
- Decision making
