# Instructions: Variables & Types

Your task is to fix the `main.go` file so that it correctly uses variables and constants.

## Objective:
1. Create a constant named `AppName` with the value `"GoStudy"`.
2. Inside `main`, declare a variable `version` of type `float64` and set it to `1.0`.
3. Create a variable `isAwesome` using the short declaration operator (`:=`) and set it to `true`.
4. The program should print all three values.

## How to Verify:
1. Navigate to `modules/02_variables`.
2. Run `go test -v`.
