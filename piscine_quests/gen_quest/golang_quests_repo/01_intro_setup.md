# Quest 01 — Go Installation & First Program

## Objectives
- Install Go
- Understand `package main`
- Understand `func main()`
- Run Go programs using `go run`

## Tasks
1. Install Go and verify with:
```bash
go version
```

2. Create a file called `main.go`
3. Print:
```text
Hello, Go!
```

4. Modify the program to print:
- Your name
- Your school
- Today's date

5. Create another program that prints a triangle using `*`.

## Bonus Challenges
- Print your output using multiple `fmt.Println()` calls
- Try using `fmt.Printf()`

## Expected Skills
- File creation
- Running Go code
- Basic terminal usage
