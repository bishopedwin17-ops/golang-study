# Quest 05 — Loops

## Objectives
- Learn loops
- Learn loop control

## Tasks
1. Print numbers 1–100.
2. Print even numbers.
3. Print odd numbers.
4. Create a multiplication table generator.
5. Find factorial of a number.
6. Generate Fibonacci numbers.

## Bonus Challenges
- Reverse a number
- Detect palindrome numbers

## Expected Skills
- for loops
- break
- continue
