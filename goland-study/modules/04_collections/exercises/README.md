# Module 04: Collections
## strlen
Write a function `StrLen(s string) int` that counts the runes in a string.
## strrev
Write a function `StrRev(s string) string` that reverses a string.
## atoi
Write a function `Atoi(s string) int` that simulates `strconv.Atoi` but returns `0` on invalid inputs.
## map_frequency
Write a function `MapFrequency(words []string) map[string]int` that counts occurrences.
