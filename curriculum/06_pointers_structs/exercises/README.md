## sortedlistmerge
You have a Linked List structure:
```go
type NodeI struct {
	Data int
	Next *NodeI
}
type List struct {
	Head *NodeI
	Tail *NodeI
}
```
Write `SortedListMerge(n1 *NodeI, n2 *NodeI) *NodeI` that merges two sorted lists.
