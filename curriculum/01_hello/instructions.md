# Instructions: Hello, Go!

Your task is to complete the `main.go` file so that it prints a specific greeting.

## Objective:
1. Define the `main` package.
2. Import the `fmt` package.
3. Create a `main` function.
4. Use `fmt.Println` to print the exact string: `"Hello, Gopher!"`

## How to Verify:
1. Open your terminal.
2. Navigate to this directory: `cd modules/01_hello`
3. Run the tests: `go test -v`

If you haven't installed Go yet, please follow these steps:
1. Download Go from [golang.org/dl](https://golang.org/dl/)
2. Install it and ensure `go` is in your PATH.
3. Restart your terminal/editor.
